<h1> E-Notes App Snapshots </h1>


E-Notes Application developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add Notes
- Edit Notes
- List Notes
- Delete Notes
- Login Model
- Registration Model
- Responsive layout

## Technology stack
- Java 8+
- JSP and Servlet
- JDBC
- MySQL database
- Eclipse IDE
- Tomcat server 8+
